# Personal Profile Site

Personal portfolio site built with Next.js. Features a cafe-menu-style project listing, experience timeline, and skills overview.

Live at: https://profile-page-sepia.vercel.app/

## Stack

- Next.js 16 (App Router) + TypeScript
- Tailwind CSS
- Framer Motion
- Playfair Display (local font)
- Deployed on Vercel

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

```bash
pnpm lint
pnpm build
```

## Content

All site content lives in `src/content/` as typed TypeScript modules. Edit those files to update profile info, projects, experience, and skills without touching any UI code.
